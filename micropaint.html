Microsoft Paint Replica

A simple drawing application inspired by Microsoft Paint, featuring basic tools like brush, eraser, color picker, shape drawing, and undo functionality, built using p5.js.

P5JS.org Link

P5.jS DEMO

Problem

Many users seek simple, intuitive drawing tools for quick sketches, annotations, or creative expression without the complexity of professional design software. While Microsoft Paint offers this functionality, it is limited to Windows platforms and lacks accessibility through the web. There is a need for a lightweight, browser-based drawing application that replicates the core features of Microsoft Paint—such as freehand drawing, shape creation, color selection, and erasing—while being accessible on any device without installation. This project aims to develop a web-based replica of Microsoft Paint using p5.js to provide users with a familiar and easy-to-use drawing experience across platforms.

Solution

This project offers a simple, browser-based drawing app built with p5.js that replicates core Microsoft Paint features like freehand drawing, shape tools, color selection, and erasing. It runs on any device without installation, providing an accessible and intuitive drawing experience for casual users.

Tools and Features

  • Color Picker
  • Free hand drawing
  • Download function
  • Eraser function
